Yesterday's Heroes

CM1040 - Web Development Final Assignment Report

Introduction
I love cars and the history of cars is an interesting subject to write about. Decide to write a story that start with the history of the first car and the revolution.
State 3 things that have inspired you when creating your website
  • Love and passion of the automobile world.
  • Growing up with my dad restoring old ladies to there peak to see that transformation.
  • wanting to share that passion with the world.
State 3 ways in which your site is accessible.
  • Render on desktop web browsers
  • Render on mobile devices
  • Each edition have a audio recording so can play and listen to the edition
State 3 ways in which you have considered usability in your site.
  • Want to keep it simple and easy to use, not over engineered.
  • Access all editions quickly from the home screen
  • Accessable on mobile
  • Content get loaded from JSON structure so even if its 5 pages the actual content get loaded using Handlebars.js and so not seperate pages per edition.
State 3 things you had to learn/find out by yourself when creating your site, and describe how you did that
  • Research on the history, finding the different images and writting the content.
  • Create all the different graphics used by the website like the logo etc.
  • Use Bootstrap 5 CSS framework to cater for responsive design and the features used by the site like the image gallery on the home page.
  • Learning how to override some Bootstrap CSS framework elements to customise.
What aspect(s) of your work do you think worked well and why?
  • Using a CSS framework - Bootstrap 5 in this case
  • Use client side Handlebars.js to load the per edition content from JSON
  • Audio that read the edition
  • Generic meta tags on the website that return the title, description and logo when site is shared to social media.
  • Responsive design. Wasn't very difficult to implement a desktop and mobile view
What aspect(s) of your work could be improved, and how might you do things differently another time?
  • Read content from a headless CMS system like Strapi.io
  • Use a paid audio service to convert text into audio to read each edition
  • Use server side pre-compiled components for Handlebars.js
  • Website use static meta tags for sharing to social media, would like it to be flexiable per edition so that when a page is shared it will have the content for the edition.
What resources were used when creating your work?
  • Bootstrap 5.0.2
  • jQuery 3.6.0
  • Wikipedia
  • w3schools Bootstrap Tutorial
  • w3schools CSS Tutorial
  • Change Bootstrap card header size
  • fetch function using jQuery
  • Black and white social icons
  • Bootstrap email contact us form
  • Bootstrap event sample
  • Canva to prepare all the images etc
  • Convert audio WAV reporting into MP3 / OGG formats
  • The audio reading of the editions was done by a family member - Yvette that have a beautiful voice.
